# InfraGenie AI - Cyberpunk Cloud Architect Dashboard

**InfraGenie AI**는 자연어로 입력된 요구사항을 분석하여 즉시 시각적인 클라우드 아키텍처 토폴로지 맵을 생성하고, 보안 취약점 진단 및 비용 기여도를 분석해 주는 사이버펑크 테마의 인터랙티브 클라우드 아키텍트 대시보드입니다.

---

## 1. 개발 목적 (Why)

* **복잡한 클라우드 인프라의 즉각적인 시각화**: AWS, Kubernetes 등 고도화된 클라우드 아키텍처 구조를 한눈에 직관적으로 파악하기 어렵다는 문제를 해결하고자 만들어졌습니다.
* **자연어 기반 신속한 프로토타이핑**: 대화형 프롬프트를 통해 인프라 디자인을 빠르게 구성하고 시각화하여, 의사결정 속도를 높이고 인프라 설계 오류를 사전에 예방합니다.
* **보안 및 비용 가시성 통합**: 설계 단계에서 예상되는 비용(Monthly Cost) 및 보안 위협 요소를 실시간으로 추적하여 "Design-by-Security" 및 "Cost-Optimization" 관점의 최적화 피드백을 제공합니다.
* **DevOps 가상 학습 및 시뮬레이션**: GitOps 에이전트의 커밋/푸시 흐름과 AI 코파일럿 대화를 결합하여, 실무 DevOps 파이프라인이 구동되는 시각적 재미와 데모 효과를 극대화했습니다.

---

## 2. 주요 기능 상세 설명 (What)

### ① 대화형 설계 프롬프트 및 프리셋 입력 (Left Panel)
* 사용자가 한글 자연어로 원하는 인프라 요구사항(예: VPC 리전, 서브넷 구성, 로드 밸런서, K8s 노드 수, 데이터베이스 다중화 등)을 작성할 수 있습니다.
* **퀵 템플릿**: 미리 최적화된 3가지 복잡한 시나리오를 원클릭으로 로드할 수 있습니다.
  * **고가용성 K8s 구조**: AWS 멀티 AZ 제어부, 워커 노드 그룹, Aurora DB 구조.
  * **Web Server & Storage**: CloudFront CDN, S3, Nginx ASG, 멀티 리전 RDS.
  * **AI 분석 & 최적화 인프라**: FastAPI, Kafka 이벤트 버퍼, GPU 워커 노드, MongoDB.

### ② 실시간 인터랙티브 토폴로지 맵 (Center Panel)
* 입력을 바탕으로 **3-Tier 구조(Public Tier, Private Tier, Secure DB Tier)**에 맞추어 네트워크 및 컴퓨팅 노드가 동적으로 배치됩니다.
* 실시간 SVG 캔버스 라인을 통해 활성화된 네트워크 커넥션과 데이터의 실시간 흐름(Glow Connection Pulse Effect)이 렌더링됩니다.
* 개별 노드를 마우스로 클릭하면, 하단 **인스펙터 패널(Inspector Panel)**에 상세 가상 IP 주소, AWS 인스턴스 스펙, 복제 노드 개수 및 가용성 진단 정보가 실시간 업데이트됩니다.

### ③ AI 인프라 분석 및 비용 최적화 리포트 (Right Panel)
* **Estimated Cost & Security Score**: 설계된 구조의 보안 점수(최대 100점)와 월간 예측 비용을 산출합니다.
* **Chart.js 연동 비용 기여도**: 네트워크, 연산, 데이터베이스, 스토리지 간의 비용 비중을 실시간 도넛 차트로 보여줍니다.
* **보안/최적화 진단 리포트**: WAF 적용 권장, S3 비암호화 타겟 경고, 스팟 인스턴스 혼합 사용 권유 등 구체적이고 실천적인 보완 조치(Actionable Insights)를 제공합니다.
* **InfraGenie Copilot 챗봇**: 인프라와 관련된 추가 요구사항(예: "보안을 강화하려면?", "비용을 더 아끼는 방법은?")을 채팅창에 질문하면 실시간 맞춤형 조언을 전달받을 수 있습니다.

## 3. 사용자가 대시보드를 활용하는 방법 (How to Use)

1. **템플릿 선택 또는 요구사항 입력**: 
   * 왼쪽 화면의 **추천 설계 템플릿** 버튼 중 하나를 클릭하거나, 입력 창에 직접 인프라 사양을 한글로 타이핑합니다.
2. **AI 인프라 설계 생성 가동**:
   * **[AI 인프라 설계 생성]** 버튼을 클릭하면 1.5초간 가상 AI 분석 게이지가 올라가며 실시간 설계 프로세스가 수행됩니다.
3. **아키텍처 토폴로지 분석 및 노드 점검**:
   * 중앙에 배치된 컴포넌트 중 세부 정보가 궁금한 노드(예: `K8s Workers NodeGroup`, `Aurora PostgreSQL Primary` 등)를 클릭하여 하단 인스펙터의 실시간 IP와 스펙을 확인합니다.
4. **비용 및 진단 보고서 검토**:
   * 우측의 비용 공유 차트 및 리포트를 검토하여 개선이 필요한 보안 위협 요소(`WARNING` 표기 등)가 있는지 식별합니다.
5. **AI 코파일럿과 추가 상담**:
   * 하단 우측 챗봇 창에 질문을 입력하여 추가 최적화 가이드라인 및 Terraform 호환 코드에 대한 설명을 제공받습니다.
